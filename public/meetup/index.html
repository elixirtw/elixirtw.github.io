Elixir Taiwan 小聚

自 2014 年起，我們每月定期舉辦實體聚會。不論你是 Elixir 新手還是資深開發者，都歡迎加入我們的交流行列！

下一場活動

報名參加小聚

時間

每月最後一個週二，19:30 開始

地點

五倍學院 / 言文字（台北市區）

費用

免費（場地飲料費自理）

Meetup 報名 Luma 行事曆

歡迎 Walk-in

無需報名也可以直接來參加！

聚會流程

輕鬆自在的交流形式，沒有笨問題，只有互動交流

1

自我介紹

19:30 開始，每個人簡單介紹自己和最近在玩的東西。有問題想討論也可以先提出來！

「我只是來聽聽」也完全 OK

2

主題分享

成員輪流分享技術主題。沒有講者時，大家一起討論最近的話題，或挑戰 AOC、Exercism 題目。

3

討論閒聊

針對分享內容深入討論，或自由分組聊天。主題不限，技術、職涯、閒聊都歡迎！

活動形式

多元的活動內容，滿足不同學習需求

主題分享

成員技術分享與經驗談

自由交流

開放式討論與問答

讀書會

一起研讀 Elixir 書籍

工作坊

實作練習與 Pair Programming

講者募集

想要分享你的經驗嗎？

不管是最近在玩的小東西、工作上遇到的有趣問題、或是任何想要分享的 Elixir 主題，我們都非常歡迎！分享的時間長度可以自由調整，5 分鐘的 Lightning Talk 或 30 分鐘的深度分享都可以。

不需要是專家，學習心得也很棒

可以用中文或英文分享

我們會協助你準備內容
