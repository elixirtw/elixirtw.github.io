為什麼選擇 Elixir?

Elixir 是一門現代化的函數式程式語言，建構在 Erlang VM 之上，專為建立可擴展、可維護的應用程式而設計

高效能並發

基於 Actor Model 的輕量級 Process，輕鬆處理數百萬個並發連線

容錯設計

「Let it crash」哲學，Supervisor 機制讓系統自動恢復，達到 99.9999999% 可用性

熱更新部署

無需停機即可更新程式碼，確保服務 24/7 不中斷運行

優雅語法

Ruby-like 的友善語法，強大的 Pattern Matching 與 Pipe Operator

Phoenix Framework

實時 Web 開發框架，LiveView 讓你用 Elixir 寫出互動式前端

活躍社群

友善的全球社群，豐富的開源套件生態系，Hex.pm 持續成長中
